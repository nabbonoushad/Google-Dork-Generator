## Google Dork Generator
*its a personal project, use on your own risk*

Main: App.java

Frame/Frontend: DorkFrame.java

Logic: DorkLogicHelper.java

Preset File: used to load keywords into the dork, can be edited from the MenuBar -> Preset -> Edit Preset

Preset Folder: Stored in AppData\Local\GoogleDorker, can be Opened from the MenuBar -> Preset -> Open Preset Location
